This is a simple Analog Clock project built using HTML, CSS and JavaScript. 
The clock shows the current time with moving hour, minute and second hands.

## Features
- Real time clock
- Smooth moving clock hands
- Clean and simple design

## Technologies Used
- HTML
- CSS
- JavaScript

## How it Works
The clock uses JavaScript to get the current time from the system and rotates the clock hands accordingly.
